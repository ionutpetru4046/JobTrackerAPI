## 🧠 Overview
Full-stack SaaS-style job tracking platform with authentication, job management, filtering, and analytics.

## ⚙️ Tech Stack
- MongoDB
- Express
- React (Vite)
- Node.js
- JWT Auth

## 🔥 Features
- Secure JWT authentication
- Full CRUD job management
- Filtering, search, pagination
- Analytics (job status breakdown)
- Protected routes (multi-tenant)
- RESTful API design
